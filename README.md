### UI & API Automation with Selenium & REST Assured

This project is a simple UI & API automation project. It is built using Selenium & Rest assured in Java



### Tech stack

* Selenium 4.34
* Rest assured 5.5
* Java: 23
* TestNG: 7.11
* Maven: 4.0
* Page Object Model (POM)

